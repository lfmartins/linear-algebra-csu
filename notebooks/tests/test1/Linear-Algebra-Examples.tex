    \section{Summary of Commands}\label{summary-of-commands}

\subsection{Matrix input}\label{matrix-input}

To input matrices, use the function \texttt{matrix\_to\_rational()}. The
argument is a two-dimensional array, as shown below:

\begin{verbatim}
A = matrix_to_rational([11,12,13],
                       [21,22,23],
                       [31,32,33]]);
\end{verbatim}

\subsection{Row operations}\label{row-operations}

To do row operations on a matrix, use the function \texttt{rop()}:

\begin{verbatim}
output_matrix = rop(input_matrix, sequence_of_operations)
\end{verbatim}

The syntax for row operations is:

\begin{itemize}
\tightlist
\item
  \texttt{\textquotesingle{}Ri*(c)+Rj=\textgreater{}Rj\textquotesingle{}}:
  Multiply row \texttt{i} by the scalar \texttt{c} and add to row
  \texttt{j}. The result is stored in row \texttt{j}.
\item
  \texttt{\textquotesingle{}Ri*(c)=\textgreater{}Ri\textquotesingle{}}:
  Multiply row \texttt{i} by the scalar \texttt{c}. The result is stored
  in row \texttt{i}.
\item
  \texttt{\textquotesingle{}Ri\textless{}=\textgreater{}Rj\textquotesingle{}}:
  Swap rows \texttt{i} and \texttt{j}
\end{itemize}

Notes:

\begin{itemize}
\tightlist
\item
  The parenthesis around the scalar \texttt{c} are always required.
\item
  The row operations must be specified as strings, that is, they have to
  be surrounded by quotes.
\end{itemize}

\subsection{Symbolic variables}\label{symbolic-variables}

To introduce symbolic variables, use the function \texttt{symbols()}.
For example:

\begin{verbatim}
x, y, z = symbols('x,y,z')
\end{verbatim}

This introduces three symbolic variables named \texttt{x}, \texttt{y},
\texttt{z}.
